# Manuscript wording — LLM-assisted coding and agreement

## Methods paragraph

Two independent LLM-assisted coding passes were conducted on the scope-screened corpus using the same frozen seven-channel codebook. The second pass was blinded to the first-pass labels, adjudicated labels, prior agreement statistics, and third-pass audit outputs. Each pass assigned one manifestation channel and a binary output-oracle observability judgment. We report raw agreement and Cohen’s kappa with bootstrap 95% confidence intervals on the unadjudicated first- and second-pass labels. Disagreements were resolved afterward using pull-request descriptions, linked issues, diffs, release notes, and regression tests; adjudication did not overwrite the raw coding records.

## Third-pass audit sentence

A separate Mistral-based LLM-assisted pass was retained as an audit artifact and analyzed separately; it was not substituted for the pre-specified second coding pass after agreement outcomes were known.

## Required disclosure

These are LLM-assisted coding passes, not human raters or human judges. The manuscript must not describe them as “two independent people,” “two annotators,” or “human inter-rater reliability.”
